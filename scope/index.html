Scope

In Javascript there are 2 types of Scope
  • Global
  • Local/Function
Scope is a dictionary into which all variables and functions are loaded. All names/identifiers must be unique In javascript, reusing a variable name/identifier overwrites the previos value - no error/silent
In javascript, thenfirst reference to a variable declares the variable, if not alreay defined - which is dangerous
functions defined using the function statement syntax 
        function myFunction() {}
are "hoisted" ie they are defined before any other code runs
functions defined using`function expression syntax 
        var myFunction = function() {}
are executed and treated like normal sequentially executed code statements
undeclared variables are auto in the global scope ie window object
